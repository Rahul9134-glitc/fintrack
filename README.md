# 💰 FinTrack Pro

A modern Personal Finance Tracker built using **HTML, CSS, and JavaScript**. It helps users manage income and expenses, visualize financial data, and store everything securely in the browser using Local Storage.

## ✨ Features

### 🔐 Authentication
- User Registration
- User Login
- Logout Functionality
- Persistent Login using Local Storage

### 📊 Dashboard
- Current Balance
- Total Income
- Total Expense
- Total Transactions

### 💳 Transaction Management
- Add Transaction
- Edit Transaction
- Delete Transaction
- Search Transactions
- Filter by Income/Expense
- Latest Transactions First

### 📈 Analytics
- Cash Flow Chart using Chart.js
- Live Dashboard Updates

### ⚙️ Settings
- Update Username
- Change Currency
- Dark Mode
- Preferences saved using Local Storage

### 🎨 User Experience
- Responsive UI
- Toast Notifications
- Modern Card Design
- Sidebar Navigation
- Modal-based Forms
